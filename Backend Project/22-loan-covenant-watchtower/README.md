# Loan Covenant Watchtower

## Summary
Loan Covenant Watchtower is a full-stack workspace for managing loan covenant workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Loan Covenant Watchtower addresses a real-world gap in financial controls, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Loan Covenant Watchtower portfolio dashboard
- Loan Covenant Watchtower trend cards
- Loan Covenant Watchtower review queue

## Backend Features
- Loan Covenant Watchtower financial data APIs
- Loan Covenant Watchtower rule evaluations
- Loan Covenant Watchtower alert workflows

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
