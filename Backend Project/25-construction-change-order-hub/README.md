# Construction Change Order Hub

## Summary
Construction Change Order Hub is a full-stack workspace for managing construction change workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Construction Change Order Hub addresses a real-world gap in project controls, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Construction Change Order Hub change board
- Construction Change Order Hub approval timeline
- Construction Change Order Hub budget impact views

## Backend Features
- Construction Change Order Hub project APIs
- Construction Change Order Hub workflow endpoints
- Construction Change Order Hub cost summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
