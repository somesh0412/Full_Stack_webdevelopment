# Clinical Trial Match Hub

## Summary
Clinical Trial Match Hub is a full-stack workspace for managing clinical trial workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Clinical Trial Match Hub addresses a real-world gap in care operations, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Clinical Trial Match Hub case dashboard
- Clinical Trial Match Hub clinical summary cards
- Clinical Trial Match Hub follow-up task views

## Backend Features
- Clinical Trial Match Hub case APIs
- Clinical Trial Match Hub status workflow endpoints
- Clinical Trial Match Hub metrics summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
