# Contract Clause Risk Reviewer

## Summary
Contract Clause Risk Reviewer is a full-stack workspace for managing contract clause workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Contract Clause Risk Reviewer addresses a real-world gap in contract review, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- Contract Clause Risk Reviewer clause review table
- Contract Clause Risk Reviewer risk cards
- Contract Clause Risk Reviewer fallback suggestions

## Backend Features
- Contract Clause Risk Reviewer document APIs
- Contract Clause Risk Reviewer review workflows
- Contract Clause Risk Reviewer decision logs

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
