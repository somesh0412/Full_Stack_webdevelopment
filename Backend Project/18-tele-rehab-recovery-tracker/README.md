# Tele-Rehab Recovery Tracker

## Summary
Tele-Rehab Recovery Tracker is a full-stack workspace for managing tele rehab workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
Tele-Rehab Recovery Tracker addresses a real-world gap in care operations, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- TeleRehab Recovery Tracker case dashboard
- TeleRehab Recovery Tracker clinical summary cards
- TeleRehab Recovery Tracker follow-up task views

## Backend Features
- TeleRehab Recovery Tracker case APIs
- TeleRehab Recovery Tracker status workflow endpoints
- TeleRehab Recovery Tracker metrics summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Medium

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
