# SME Carbon Ledger

## Summary
SME Carbon Ledger is a full-stack workspace for managing sme carbon workflows with live dashboards, seeded operational records, and room to extend into production features.

## Problem
SME Carbon Ledger addresses a real-world gap in emissions accounting, where teams still rely on spreadsheets, email, or delayed follow-up.

## Frontend Features
- SME Carbon Ledger impact dashboard
- SME Carbon Ledger evidence uploads
- SME Carbon Ledger reduction planner

## Backend Features
- SME Carbon Ledger factor mapping APIs
- SME Carbon Ledger reporting endpoints
- SME Carbon Ledger scenario summaries

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express
- PostgreSQL-ready schema

## Difficulty
Hard

## Run Locally
1. In `backend`, run `npm install` and then `npm run dev`.
2. In `frontend`, run `npm install` and then `npm run dev`.
3. Open the frontend URL shown by `serve`.

## API Endpoints
- `GET /api/health`
- `GET /api/overview`
- `GET /api/work-items`
- `GET /api/metrics`
- `POST /api/work-items`
